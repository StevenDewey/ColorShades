Description:

One of the color theories in web page design is to use monotone chromatic colors. A monotone color scheme is one hue with different saturations and shades. In other words, a blue monotone color scheme contains various shades of the same blue, from light blue to dark blue. Web sites that solely use the same monotone color scheme can be dull, but used correctly, monotone works really well. The web page designers that work in your company have asked you to create a monotone color scheme calculator. Given a source color, they want to see one shade lighter and one shade darker. HTML colors are given in the form #RRGGBB, where RR is the red part in hexadecimal (00-FF), GG is the green part, and BB is the blue part.

Your final project is to create a small web site that calculates darker and lighter versions of colors for developers. I've already started a project for you (this project), and it should run correctly right now. It already includes JQuery and Bootstrap. If it doesn't run correctly, stop here and make it work. This won't count against your time.

Some of the requirements you'll see below could be done with Javascript or with CSS. However, you need to do exactly what you are asked to do below. This isn't the time to deviate from the instructions, and it isn't the time to impress me with extra features. Just go through the instructions and follow them.

The Tasks (start your time now):

  1. Make the following modifications to the base template:
    • Add a title (both the title tag and an h1 tag) that prints on all pages of the site.
    • Add an image of your choosing underneath the h1 tag. Place this image in the homepage/media/ folder.

  2. Create base.css in your styles/ folder:
    • Style your image to be 200 x 200 pixels.
    • Center both the page title (h1 tag) and the image horizontally.

  3. Create select.py in your views/ folder:
    • Add a view_function called process_request. This function should create and clean the color form. If the form is valid, redirect the user to a the /homepage/color/RRGGBB/ url, where RRGGBB is the hex code the user entered.
    • Add a form class with one text input field. The user will enter the color here. Place one clean method in the form class that ensures the user entered a valid color in hex format (RRGGBB).

  4. Create select.html in your templates/ folder:
    • Place the form on the page (the variable comes from select.py).
    • Add the form tags and submit button.

  5. Create color.py in your views/ folder, and add the following:
    • Add one or more functions to scale colors to darker/lighter shades. I suggest you use the code found at http://thadeusb.com/weblog/2010/10/10/python_scale_hex_color.
    • Add a view_function called process_request:
      • Get the color code from the urlparams. You can assume that the code is valid (it was cleaned on the select page).
      • Create a list of 5 items with two lighter shades of the color, the exact color, and two darker shades of the color. The colors in the list should get progressively lighter (start with dark and go lighter).
      • For example, the color #df3c3c might go to [ #852424, #b23030, #df3c3c, #ff4848, #ff5454 ]. This example used scale values of .6, .8, 1, 1.2, and 1.4.
      • Send this list variable to the color.html template.

  6. Create color.html in your templates/ folder:
    • Using a for loop, go through the five colors in the list (that you created in color.py) and print five boxes on the screen. Set the background color of each box to each color, and print the color hex code inside each box.
    • You are welcome to use any HTML tags for your boxes, from table cells to divs.
    • Set the color and box width/height in the style property of each box tag. Each box should be about 200 x 200 pixels.
    • Provide a bootstrap-styled "back" button that takes the user back to the /homepage/select/ page.

  7. For the final 10% of the final, add an ajax call to select a random color code for the user. If you don't get to this part, you'll still get 90% of the points.
    • Add a bootstrap-styled button to the select.html template.
    • When this button is pressed, trigger an Ajax call to /homepage/select.random_color/
    • The random_color() view function in select.py should create a random hex color and return it to the select page. Use Python's random library (it's built into python, so just "import random") to generate a hex code between 000000 and ffffff.
    • When the random_color() method returns successfully, place the random code in the form's input box.
